Context 2 / 5

👤 End-User

  • Home cooks and food enthusiasts
  • Students who want quick meal ideas
  • Families organizing their recipe collection
  • Anyone wanting to find recipes based on ingredients they already have in the fridge

❌ The Problem

People struggle to:

  • Save and organize recipes in one place
  • Find what to cook with available ingredients
💡 Product Idea:
A Telegram bot that lets users save, manage, and discover recipes using Integrated AI for smart ingredient matching with graceful fallback.

➕ Save

Recipes with ingredients & step-by-step instructions

🔍 Discover

Find recipes by entering ingredients you have

🧠 Smart

Handles typos & synonyms automatically

Implementation 3 / 5

🛠 Tech Stack

Python aiogram PostgreSQL Docker Integrated AI

📦 Version 1 — Core Bot

  • Basic functions: adding, viewing, editing, and deleting recipes
  • Ingredient quantities parsing (e.g., "4 pcs", "200g")
  • Inline keyboard for edit/delete selection

📦 Version 2 — Smart Matching + Integrated AI

  • Integrated AI corrects typos in real-time ("tomatoe" → "tomatoes")
  • Auto synonym expansion ("tomatoes" → also matches "tomato")
  • Partial matching (LIKE) for ingredient search
  • Fuzzy fallback via Python difflib
  • Graceful fallback — bot works without AI

🏗 Architecture

User → Telegram → aiogram Bot → AI Service → PostgreSQL

Bot polls Telegram for messages, corrects typos via AI, then queries the database

🚀 Deployment

Docker Compose on VM — bot + database, auto-restart, health checks

🔑 Key Achievement:
Integrated AI fixes typos like "tomatoe" → "tomatoes" automatically, with built-in fallback
